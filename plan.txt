Scope
─────
Tests live in #[cfg(test)] mod tests blocks inside each source file.
All tests compile under `cargo test --no-fail-fast`.
GUI rendering tests (egui widgets) are out of scope — they require a
headless display server. Pure functions in GUI modules ARE tested.
Network multicast tests are out of scope — environment-dependent.
main.rs is not tested directly — entry points require process::exit stubs.
